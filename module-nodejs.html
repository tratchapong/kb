üìÅ File Extensions & package.json

Node.js determines the module type based on either:

  • The file extension:

    • .js = CommonJS by default (unless "type": "module" in package.json)

    • .cjs = Always CommonJS

    • .mjs = Always ES Module

  • Or the package.json:

    json
    {
  "type": "module"
}

    This treats .js files as ES Modules.
